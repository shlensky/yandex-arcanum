# Домашнее задание для ШРИ-2019 (Москва)

Сделать React приложение, которое будет показывать структуру файлов и директорий репозитория.

## Установка

    npm install
    cp .env.example .env

## Разработка

    npm start

Сервер будет доступен по адресу [http://localhost:7777/](http://localhost:7777/)
    
## Продакшен сборка

    npm run build
